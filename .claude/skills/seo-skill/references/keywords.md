## Competitor Gap Analysis

These are the platforms users compare us to. None of them teach Izon — this
is Beeli's most defensible advantage.

| Competitor | Languages | Gap | Our Angle |
|---|---|---|---|
| Duolingo | ~40 languages — no Izon, no Niger Delta languages | Zero African minority languages | "Finally learn Izon — the language Duolingo forgot" |
| Babbel | ~15 European-focused languages | No African languages at all | Heritage language angle Babbel cannot serve |
| Drops | Some African languages (Swahili, Yoruba) | No Izon, no Niger Delta | We serve the languages Drops skipped |
| uTalk | Broad coverage, Yoruba included | No Izon | Deeper, more authentic than a phrasebook app |
| Pimsleur | 50+ languages — no Izon | Audio only, no interactive features | More engaging with quizzes, journaling, community |
| YouTube tutorials | Izon content exists but scattered | No structured curriculum | Structured lessons vs. scattered YouTube videos |

**Our edges over every competitor:**
- Only platform in the world with structured Izon lessons
- Audio-first approach from native speakers
- Built-in dictionary, quiz, and matching games
- Community/social learning layer (feed, journaling)
- Educator tools (classroom panel, content flagging)
- Multi-language UI (en/fr/pcm) for diaspora learners globally
- Contribution system to involve the community in building the language corpus
